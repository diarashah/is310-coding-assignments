# The Social Media Feed Maze
Diara Shah

## How to Solve
This maze explores how social media platforms structure attention and visibility.

## Helpful Commands
- ls
- ls -la
- cd
- cat

## Instructions
1. Start in the `maze/start` directory.
2. Read clues using `cat`.
3. If something seems missing, check for hidden files.
4. The maze ends in `maze/exit/clues.txt`.
